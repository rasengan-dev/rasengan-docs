# defineRoutePage

The `defineRoutePage` is a function that allows you to define `metadata` of a `page`.

<Tabs items={['TypeScript', 'JavaScript']} defaultIndex="1">
  <Tabs.Tab>
    ```tsx filename="home.page.tsx" {13-17} showLineNumbers copy
    import { PageComponent, defineRoutePage } from "rasengan";

    class Home extends PageComponent {
      render() {
        return (
          <div>
            <h1>Home Page</h1>
          </div>
        )
      }
    }

    export default defineRoutePage({
      path: "/",
      title: "Home",
      description: "This is the home page",
    })(Home);
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```jsx filename="home.page.jsx" {13-17} showLineNumbers copy
    import { PageComponent, defineRoutePage } from "rasengan";

    class Home extends PageComponent {
      render() {
        return (

            <h1>Home Page</h1>
          </div>

      }
    }

    export default defineRoutePage({
      path: "/",
      title: "Home",
      description: "This is the home page",
    })(Home);
    ```
  </Tabs.Tab>
</Tabs>

This function takes an object with the following properties:

| Properties    | Type     | Description                  | Default    | Optional |
|---------------|----------|------------------------------|------------|----------|
| `path`        | `string` | The path of the page.        | -          | No       |
| `title`       | `string` | The title of the page.       | Class name | Yes      |
| `description` | `string` | The description of the page. | -          | Yes      |


And returns a function that takes a `PageComponent` and returns a new `PageComponent` with the `metadata` attached to it.
