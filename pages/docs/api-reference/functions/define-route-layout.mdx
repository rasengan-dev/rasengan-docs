# defineRouteLayout

The `defineRouteLayout` is a function that allows you to define the base `path` of the pages route under a `layout`.

<Tabs items={['TypeScript', 'JavaScript']} defaultIndex="1">
  <Tabs.Tab>
    ```tsx filename="app.layout.tsx" {15-17} showLineNumbers copy
    import { LayoutComponent, defineRouteLayout, Outlet } from "rasengan";

    class AppLayout extends LayoutComponent {
      render() {
        return (
          <div>
            <h1>App Layout</h1>

            <Outlet />
          </div>
        )
      }
    }

    export default defineRouteLayout({
      path: "/",
    })(AppLayout);
    ```

  </Tabs.Tab>
  <Tabs.Tab>
    ```jsx filename="app.layout.jsx" {15-17} showLineNumbers copy
    import { LayoutComponent, defineRouteLayout, Outlet } from "rasengan";

    class AppLayout extends LayoutComponent {
      render() {
        return (
          <div>
            <h1>App Layout</h1>

            <Outlet />
          </div>
        )
      }
    }

    export default defineRouteLayout({
      path: "/",
    })(AppLayout);
    ```

  </Tabs.Tab>
</Tabs>

This function takes an object with the `path` property, and returns a function that takes a `LayoutComponent` as an argument which finally returns a new `LayoutComponent` with the defined path.
